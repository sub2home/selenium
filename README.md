### Installation
You need have Ruby2 installed for the tests to run. If you haven't [bundler](http://bundler.io/) installed yet, run:

```sh
$ gem install bundler
```

The following installs all you need.
```sh
$ bundle install
```

### Run tests
```sh
$ rspec
```
